Models¶

This is the Carousel models module that contains definitions for the Model class.

The Carousel model contains five layers: Data, Formulas, Calculations, Outputs and Simulations. The Data layer organizes Data Sources by providing methods to add and load data for Carousel. The Formulas layer loads Formulas used by Calculations calculations. The Outputs layer organizes the calculated outputs for use in other calculations. Finally the Simulations layer organizes options such as how long the simulation should run and takes care of actually running the simulation.

Model Base¶

class carousel.core.models.ModelBase[source]¶

Base model meta class. If model has class attributes “modelpath” and “modelfile” then layer class names and model configuration will be read from the file on that path. Otherwise layer class names will be read from the class attributes.
