Getting Started¶

Requirements¶

  • quantities
  • numpy
  • xlrd
  • nose
  • sphinx

Installation¶

You can use pip or disutils to install circus.

$ pip install circus

$ curl -Ok https://github.com/SunPower/Circus/archive/v0.1.tar.gz
$ tar -xf v0.1.tar.gz
$ cd Circus-0.2
$ python setup.py install

Quickstart¶

Circus adds the script circus-quickstart.py to quickly start a project.

$ circus-quickstart.py MyCircusProject.

This creates a new folder for MyCircusProject with seven sub-folders.

MyCircusProject
|
+-+- mycircusproject
| |
| +- __init__.py
|
+- models
| |
| +- default.json
|
+- simulation
|
+- outputs
|
+- calculations
|
+- formulas
|
+- data

Go through the examples in the Tutorials to implement your first Circus model.
